
=================================================
ElectroShop DZ
=================================================

IMPORTANT

Ce projet contient un backend développé avec
Node.js et Express.

Pour exécuter toutes les fonctionnalités
(inscription, connexion et commandes),
veuillez suivre les étapes suivantes :

1. Ouvrir un terminal

2. Aller dans le dossier backend :

cd backend

3. Installer les dépendances :

npm install

4. Démarrer le serveur :

node server.js

5. Vérifier l'affichage du message :

Backend ElectroShop DZ fonctionne !

6. Ouvrir ensuite index.html
dans le navigateur.

=================================================
Remarque
=================================================

Le dossier node_modules n'est pas inclus dans
le dépôt GitHub car il est généré automatiquement
par la commande :

npm install
