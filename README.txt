ELECTROSHOP DZ

Projet de Développement Web

==================================================
DESCRIPTION DU PROJET
=====================

ElectroShop DZ est un site e-commerce développé dans le cadre du module de Développement Web.

Le site permet aux utilisateurs de :

* Créer un compte.
* Se connecter à leur compte.
* Parcourir les produits disponibles.
* Rechercher des produits.
* Filtrer les produits par catégorie.
* Ajouter des produits au panier.
* Modifier les quantités du panier.
* Passer une commande.
* Enregistrer les commandes.

==================================================
BACKEND
=======

Le projet contient un backend développé avec Node.js et Express.

Le backend gère :

* L'inscription des utilisateurs.
* La vérification des identifiants de connexion.
* L'enregistrement des commandes.
* Le stockage des données dans des fichiers JSON.

Fichiers utilisés :

* users.json
* orders.json

==================================================
INSTRUCTIONS D'EXÉCUTION
========================

1. Ouvrir le dossier backend.

2. Installer les dépendances :

npm install

3. Lancer le serveur :

node server.js

4. Vérifier que le message suivant apparaît :

Backend ElectroShop DZ fonctionne !

5. Ouvrir ensuite le fichier index.html dans le navigateur.
